# System Skill

> Shell commands, process management, system info.

## Actions

| Action | Description | Params |
|--------|-------------|--------|
| `exec` | Run a PowerShell command | `--command "whoami" [--timeout 30]` |
| `info` | System info (hostname, OS, user) | |
| `processes` | List top processes by CPU | `[--limit 20]` |
| `env` | Get environment variable | `--name PATH` |

## Requirements

- PowerShell 5.1+
